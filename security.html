Security

Security Policy

The security of the Stremax programming language and its ecosystem is our top priority. We are committed to ensuring the safety and reliability of all smart contracts developed using Stremax.

Reporting a Vulnerability

If you discover a security vulnerability, please report it immediately to security@stremax.io. We take all reports seriously and will respond within 24 hours.

Security Process

  1. Vulnerability Reporting

    Report security issues to our dedicated security team

  2. Assessment

  3. Resolution

    Rapid development and testing of security fixes

    Responsible disclosure after patch deployment
